# CLI Usage

Learn how to use the command-line interface.

## Basic Commands

Start the development server:

```bash
my-package dev
```

Build for production:

```bash
my-package build
```

## Configuration

Create a configuration file:

```bash
my-package init
```

## Advanced Usage

Deploy to production:

```bash
my-package deploy --env production
```

Generate types:

```bash
my-package generate-types
```
